API Documentation

General instructions

The simplest example of a synchronous request with parameters would be something like:

curl --request 'POST' \
    --data '{"delay_s":0.5}'  \
    https://faas.srv.disarm.io/function/longrun

There are detailed instructions for running the functions in development (e.g. using curl) and in production (e.g. from Python). For more general instructions and background information, see the project wiki.
